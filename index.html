يعني إيه HTML؟

تخيّل معايا إنك بتبني بيت.

أول حاجة بتبدأ بيها هي الأساس: طوب، حيطان، شبابيك، أبواب. لسه مفيش ألوان، مفيش ديكور، بس الهيكل واقف... صح؟

الـ HTML هي الأساس ده في المواقع. هي اللي بتحددلك:

  • <h1> ده عنوان
  • <p> دي فقرة
  • <img> دي صورة
  • <button> ده زرار
  • <form> ده نموذج تسجيل

HTML = أساس البيت

بتعمل إيه بالظبط؟

HTML مسؤولة عن شكل الصفحة من ناحية الهيكل

لكن مش مسؤولة عن الألوان ولا الحركة.

بمعنى:

  • HTML = الأساس والطوب
  • CSS = الدهان والتجميل
  • JavaScript = الحركة والتفاعل

يعني الزرار في HTML ثابت، CSS تخلّيه لونه أحمر، JavaScript تخليه يطير 😄
بس لازم تبدأ بـ HTML.

مثال عملي: زرار في موقع

HTML

<button id="myButton">
  اضغط هنا
</button>

تحدد هيكل الزرار

CSS

button {
  background: red;
  color: white;
  border-radius: 8px;
}

تجمّل شكل الزرار

JavaScript

myButton.onclick = function() {
  alert("تم الضغط!");
  this.style.transform = "rotate(360deg)";
}

أول كود HTML ليك

دلوقتي هنكتب أول صفحة HTML ونشغلها بنفسنا 💪

افتح أي محرر كود زي VS Code، واكتب الكود اللي هتشوفه على اليمين.

الكود ده بيمثل هيكل أساسي لأي صفحة HTML، وهو اللي هيظهر للمستخدم في المتصفح.

ملاحظة مهمة: لاحظ إن كل علامة (Tag) ليها بداية وليها نهاية:

  • <html> بداية الصفحة و </html> نهايتها
  • <head> بداية رأس الصفحة و </head> نهايته
  • <body> بداية جسم الصفحة و </body> نهايته
<!DOCTYPE html> <html>   <head>     <title>أول صفحة HTML</title>   </head>   <body>     <h1>أهلاً بيك!</h1>     <p>دي أول صفحة بكتبها بإيدي بلغة HTML ❤️</p>   </body> </html>

ملحوظة: أي تعديل هتعمله في الكود، احفظ الملف وحدّث الصفحة في المتصفح (F5) عشان تشوف التغييرات.

<html>
  <head>
    <title>أول صفحة HTML</title>
  </head>

    <h1>أهلاً بيك!</h1>

  </body>
</html>

ليه بنستخدم index.html؟

اسم index.html هو الاسم الافتراضي اللي بيدور عليه المتصفح أول ما تفتح أي موقع.
يعني لما تكتب www.google.com، المتصفح بيدور على ملف اسمه index.html في سيرفر جوجل.

أسئلة بتتلخبط فيها

هل HTML لغة برمجة؟

لأ، هي مش برمجة، دي "لغة توصيف"، بتنظم بس شكل الصفحة. مش بتقدر تعمل عمليات حسابية أو شروط أو حاجات معقدة زي لغات البرمجة.

هل HTML كفاية؟

لوحدها لأ، بس هي أول خطوة... مينفعش تبدأ CSS أو JS من غير HTML. هي الأساس اللي بتبني عليه باقي التقنيات.

HTML صعبة؟

لأ أبدًا، هي من أسهل اللغات اللي ممكن تتعلمها. مجرد علامات (tags) بتحدد نوع المحتوى، وقواعدها بسيطة جدًا.

نصائح مهمة

  • ابدأ بـ HTML قبل أي حاجة تانية
  • اكتب كود نظيف ومنسق عشان تقدر تفهمه بعدين
  • استخدم محرر أكواد زي VS Code لتسهيل الكتابة
  • تأكد إن كل tag مفتوح ليه tag مقفول
